# Wavelet-HAR-Reconciliation-data
This is the data used in the paper 'Enhancing volatility prediction: A wavelet-based hierarchical forecast reconciliation approach' in the Journal of Forecasting.

Each data file contains dates and 30 columns correpsonding to the 30 stocks

- **Realized Volatility (RV) data file:** `rv_stocks.csv`.
- **Daily returns data file:** `rets_stocks.csv`.
- **Partial Variances 1-3 (PV) data files:** `pvar1_stocks.csv`, `pvar2_stocks.csv` and `pvar3_stocks.csv`.
- **Realized Semi-Variances - positive(p) and negative (n) (PV) data files:** `rsv_p_stocks.csv`, `rsv_n_stocks.csv`
